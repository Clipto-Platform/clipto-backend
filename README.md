# Clipto backend

Backend services
- twitter verifications
- file uploads to ARweave
- metadata uploads to ipfs

## Setup
1. Copy `.env` file
```
cp .env.example .env
```

2. Update credentials in `.env` file

3. Run the server
```
npm install
npm run start
```

4. Running end-to-end tests
```
npm run test:e2e
```

5. All available commands
```
npm run
```
