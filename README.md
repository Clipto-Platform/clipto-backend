# Clipto backend

Does Twitter verification, accepts files & uploads to arweave.

## Setup
1. Copy `.env` file
```
$ cp .env.sample .env
```

2. Update credentials in `.env` file
3. Run the server
```
$ npm install
$ npm run start
```
4. Running end-to-end tests
```
$ npm run test:e2e
```
5. All available commands
```
$ npm run
```
